## Title

Show HN: lever-runner — A shell AI where prompt injection is physically impossible

## Post

Every AI shell tool works the same way: ship your terminal context to a cloud LLM, get a command back. Copilot CLI burns ~2,000 tokens per query. Warp AI uses ~3,500. The LLM sees your filesystem, your env vars, your history — because it has to synthesize commands from scratch every time.

lever-runner doesn't do that.

The insight: you don't need an LLM at runtime. You need it once, to understand what you mean. After that, it's vector search.

The architecture is three gates:

1. **Rust fastloop** (50µs) — template match. "check disk" → `df -h`. Most common queries resolve here.
2. **Python embedding cache** (200µs) — cosine similarity against your taught commands. 44% hit rate in production.
3. **LLM** (500ms) — the fallback. It sees exactly one thing: a 5-word intent phrase. It returns a 3-8 word phrase. That's it.

Gate 3 is the only one that costs money or sends data anywhere. The LLM never sees your shell, your files, or your commands. It sees "show disk usage" and returns "show disk usage". Prompt injection is physically impossible — there's no surface to inject into. The command isn't generated by the LLM; it's looked up from a pre-approved table via cosine search.

The entire LLM system prompt is 58 tokens:

```
You compress a user request into a short verb-noun phrase of 3-8 words.
Output ONLY the phrase, lowercase, no punctuation, no quotes, no prefix.
```

Add ~10 tokens for the user message, ~8 for the response. That's ~76 tokens per query. Compare to 2,000-5,000 for function-calling agents.

Or skip the LLM entirely. Passthrough mode (`LLM_BACKEND=passthrough`) uses your exact words as the search key. Zero tokens. $0/month. No data leaves the machine. 67 built-in commands cover system, docker, git, and networking out of the box.

Real benchmarks on actual hardware (Ryzen 5900X / RTX 4050):

- Vector search p50: 7.6ms
- Template match: 1.7µs
- Teach throughput: 122 commands/sec
- 202 tests passing, 216 skill pack commands

Teach it once, it runs forever:

```
lever teach "show my ip" --command "curl ifconfig.me"
lever "what's my ip"  # → curl ifconfig.me
```

The stack runs from metal to cloud: CUDA → OpenCL → NEON → C → Rust → WASM → Python. ARM64 supported — works on Oracle Cloud Free Tier and Raspberry Pi.
